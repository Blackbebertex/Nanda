# Phase 1 — Discovery and Design

**Duration:** 2–4 weeks

## Activities
1. Map customer journeys.
2. Define personas.
3. Define data sources.
4. Draft consent flows.
5. Define recommendation rules.
6. Define advice-boundary taxonomy.
7. Create UI wireframes.
8. Define API contracts.
9. Define audit schema.
10. Define AI guardrail policy.

## Deliverables
- PRD
- SRS
- UX wireframes
- Consent flow
- Data dictionary
- API specification
- Recommendation rule catalogue
- Compliance policy matrix
- Threat model
- Test strategy
