# Stakeholder Map
- **Retail Customer:** Uses ARTHA for guidance.
- **Relationship Manager:** Receives escalated cases.
- **Product Owner:** Owns roadmap and metrics.
- **Engineering Team:** Builds the system.
- **Compliance Team:** Approves boundaries and disclosures.
- **Risk Team:** Reviews suitability and models.
- **Legal Team:** Reviews terms and consent.
- **InfoSec Team:** Threat models and data protection.
