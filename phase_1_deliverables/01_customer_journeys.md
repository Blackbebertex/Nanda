# Customer Journeys
**Primary Journey: Monthly Wealth Check-In**
1. Customer opens bank app and taps ARTHA.
2. ARTHA validates session and consent status.
3. System loads financial snapshot and computes signals (savings rate, etc.).
4. ARTHA provides personalized insights (e.g., dormant asset detection).
5. Customer asks for details, triggering explainability card.
6. If advice boundary is crossed, ARTHA escalates to RM handoff flow.
