# Phase 0 — Project Initiation

**Duration:** 1–2 weeks

## Activities
1. Confirm business sponsor.
2. Confirm product owner.
3. Confirm compliance and risk reviewers.
4. Define MVP scope.
5. Define success metrics.
6. Approve regulatory positioning.
7. Finalise project governance.
8. Create delivery backlog.
9. Approve architecture baseline.
10. Approve data access approach.

## Deliverables
- Project charter
- Business case
- MVP scope document
- Stakeholder map
- Risk register
- Initial product backlog
- Regulatory-positioning note
- High-level architecture
