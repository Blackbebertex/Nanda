# Project Charter: ARTHA AI

**Sponsor:** Retail Banking Head
**Objective:** Deploy a bank-native, consent-first wealth guidance copilot embedded inside the bank’s mobile application.
**Key Milestones:**
- Phase 0: Project Initiation (Weeks 1-2)
- Phase 1: Discovery and Design (Weeks 2-4)
- Phase 2: MVP Build (Weeks 6-10)
- Phase 3: Testing and Validation (Weeks 3-5)
- Phase 4: Pilot Launch (Weeks 8-12)
- Phase 5: Production Rollout (Months 3-6)
